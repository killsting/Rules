### 规则介绍

一直没有写过介绍，那也正式介绍下规则优点，其实用上了应该就很了解功能：

* 爱奇艺过滤首家发布；
* APP启动广告首家发布；
* 更快速智能的分流；
* 两种规则适配更灵活；
* 屏蔽统计站点获取数据；
* 规避商家、站点、软件隐私获取；
* 屏蔽运营商web页面浏览流量提醒；
* 自定义DNS方案抗干扰；
* 黑白名单更快速识别；
* 各大网盘支持;

### 规则区分

[全能规则]
包含黑白名单，适配所有机型;

[精简规则]
包含白名单,节省内存占用适配越狱机型；
